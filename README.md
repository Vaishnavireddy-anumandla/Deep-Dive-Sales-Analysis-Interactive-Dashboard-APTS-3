# 📊 Deep Dive Sales Analysis & Interactive Dashboard

## 📌 Project Overview
This project focuses on performing a deep-dive analysis of a retail sales dataset and building an interactive dashboard to uncover meaningful business insights. The goal is to support data-driven decision-making by analyzing sales performance across regions and product categories.

---

## 🎯 Objectives
- Define key business KPIs
- Perform deep-dive analysis on sales data
- Build an interactive dashboard
- Generate actionable business insights

---

## 📂 Dataset
The dataset contains the following fields:
- Order ID
- Order Date
- Customer ID
- Region
- Category
- Sales
- Profit
- Quantity

---

## 📈 Key Performance Indicators (KPIs)
- **Total Sales** = Sum of all sales  
- **Total Profit** = Sum of all profit  
- **Total Orders** = Count of unique orders  
- **Average Order Value** = Total Sales / Total Orders  

---

## 🔍 Deep-Dive Analysis
A segmentation analysis was performed to understand:
- Sales performance across different regions  
- Category-wise contribution to total revenue  
- Sales trends over time  

---

## 📊 Dashboard Features
- KPI Cards for quick overview  
- Bar Chart: Sales by Region  
- Pie Chart: Sales by Category  
- Line Chart: Sales Trend  
- Interactive Filters (Region, Category)  

---

## 💡 Key Insights
- West region has the highest sales performance  
- Technology category contributes the most revenue  
- Sales trend shows a gradual decline over time  
- Furniture and Office Supplies contribute less compared to Technology
